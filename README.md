# Diabetes Prediction – Pima Indians Dataset

**Goal**: Predict diabetes (yes/no) using 8 clinical features.

**What I did**:
- Cleaned invalid zeros in medical data
- Visualized data with histograms and boxplots
- Trained Random Forest and Logistic Regression
- Evaluated models with accuracy, confusion matrix, and feature importance
- Added my own interpretations and color changes

**Skills**: pandas, seaborn, matplotlib, scikit-learn, classification, EDA

**Dataset**: UCI Machine Learning Repository – Pima Indians Diabetes  
Link: https://archive.ics.uci.edu/dataset/34/diabetes

Note: This is my self-taught project. All interpretations and changes are my own work.
